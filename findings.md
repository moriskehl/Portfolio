# Findings & Decisions

## Requirements
- Premium Landing Page (Software Developer & Skier theme).
- Project display: iframes for sub-projects + direct access via routes.
- Resume page with images.
- Achievements section (e.g., 2nd place German Ski League).
- Complex/scalable folder structure.
- User retains full control over design/execution.
- Hosted on DigitalOcean (informational for now).

## Research Findings
- Modern portfolio trends: Glassmorphism, subtle micro-animations, high-quality typography (Inter/Outfit).
- Monorepo-lite structure allows independent projects to live alongside the main site.

## Technical Decisions
| Decision | Rationale |
|----------|-----------|
| Vanilla CSS | Maximum control and visual excellence without framework overhead. |
| Structured /projects folder | Allows iframing and direct serving of sub-projects. |
